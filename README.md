# IMDB Clone

![React](https://img.shields.io/badge/React-18.2.0-blue)
![Node](https://img.shields.io/badge/Node-required-green)

## ğŸ’» Cara Menjalankan

### Development Mode

Jalankan aplikasi dalam mode development:
```bash
npm start
```

Buka [http://localhost:3000](http://localhost:3000) untuk melihat aplikasi di browser.

Halaman akan otomatis reload ketika Anda membuat perubahan pada kode.

### Production Build

Build aplikasi untuk production:
```bash
npm run build
```

Perintah ini akan membuat folder `build` yang berisi versi optimized dari aplikasi yang siap di-deploy.

### Testing

Jalankan test runner:
```bash
npm test
```
